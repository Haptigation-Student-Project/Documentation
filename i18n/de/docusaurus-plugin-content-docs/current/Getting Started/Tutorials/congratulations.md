# Herzlichen Glückwunsch!

Sie haben gerade die **Grundlagen von Docusaurus** gelernt und einige Änderungen an der **Ausgangsvorlage** vorgenommen.

Docusaurus hat **noch viel mehr zu bieten**!

Haben Sie **noch 5 Minuten**? Werfen Sie einen Blick auf **[Versionierung](../tutorial-extras/manage-docs-versions.md)** und **[i18n](../tutorial-extras/translate-your-site.md)**.

Ist etwas **unklar** oder **fehlerhaft** in diesem Tutorial? [Bitte melden Sie es!](https://github.com/facebook/docusaurus/discussions/4610)

## Was kommt als Nächstes?

- Lesen Sie die [offizielle Dokumentation](https://docusaurus.io/)
- Ändern Sie Ihre Site-Konfiguration mit [`docusaurus.config.js`](https://docusaurus.io/docs/api/docusaurus-config)
- Fügen Sie "Navbar"- und "Footer"-Elemente mit [`themeConfig`](https://docusaurus.io/docs/api/themes/configuration) hinzu
- Fügen Sie ein benutzerdefiniertes [Design und Layout](https://docusaurus.io/docs/styling-layout) hinzu
- Fügen Sie eine [Suchleiste](https://docusaurus.io/docs/search) hinzu
- Finden Sie Inspiration im [Docusaurus Showcase](https://docusaurus.io/showcase)
- Engagieren Sie sich in der [Docusaurus Community](https://docusaurus.io/community/support)
